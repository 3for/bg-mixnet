## In one sentence

Implementation of the Bayer-Groth mixnet with non-interactive Toom-Cook optimisation using Keccak SHA-256 hash functions.

### Dependencies

- make
- gcc
- NTL library

### Build

make

### Configure

Modify the parameters in the `config` file.

### Execute

./shuffle
